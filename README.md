# FitTrack-JS


FitTrack is a modern fitness tracking web application that helps users monitor workouts, track progress, and stay consistent with their fitness goals. Built using web technologies, it provides an intuitive and responsive user experience.

🚀 Features
📊 Track daily workouts and activities
📈 Monitor progress over time
🔥 Clean and responsive UI
⚡ Fast performance with modern web stack

🛠️ Tech Stack
Frontend: HTML, CSS, JavaScript
APIs: Fetch-based data handling
